Página de Ejemplo con Widget de Chat

Esta es una página HTML simple que demuestra cómo integrar un widget de chat cargado desde local.

Nota: El widget de chat aparecerá en la esquina inferior derecha de la página como un botón flotante. Haz clic en él para abrir la ventana de chat.

Características del Widget

  • Interfaz de chat intuitiva y moderna
  • Animaciones de escritura para respuestas del bot
  • Personalizable mediante configuración
  • Integración con n8n a través de webhooks

Configuración del Widget

El widget está configurado para conectarse al siguiente webhook:
